## Usage

### Install

```
yarn install
```

### Develop

```
yarn dev
```

### Test

```
yarn test
```

### Deploy

```
yarn deploy
```

### Create Database

Create a D1 database called `notifications`

```
yarn db:create
```

### Env vars and schema

Set the environment variables in [wrangler.toml](wrangler.toml)

Apply the [schema file](/schemas/schema.sql)

```
yarn db:schema
```
